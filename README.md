# Ultimate Shower Experience 

Bienvenue dans le meilleur simulateur de douche en réalité virtuelle !

Apprenez à doucher un pingouin en suivant les instructions au mur.

Pour cliquer sur un bouton, utilisez la gachette avant sur la manette, et pour attraper des objets, utilisez la gachette sur le côté de la manette.
